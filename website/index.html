Zero-Dependency TypeScript-First Fullstack Framework

Synapse combines a high-performance Rust compiler with a Node.js TypeScript runtime, enforcing best practices and TDD with zero escape hatches. Build universal applications that work anywhere.

synapse-app.ts
import { SynapseFramework } from '@snps/core';
import { Button, Card } from '@snps/ui';

const app = new SynapseFramework({
  name: 'My App',
  version: '1.0.0'
});

const button = new Button({
  children: 'Click me',
  variant: 'primary',
  onClick: () => console.log('Hello World!')
});

app.render(button);

Why Choose Synapse?

Built for developers who demand excellence, Synapse enforces best practices and provides the tools you need to build exceptional applications.

Zero Dependencies

No external dependencies means faster builds, smaller bundles, and complete control over your application.

TypeScript First

Built from the ground up for TypeScript with strict type checking and comprehensive type definitions.

TDD Enforced

Test-Driven Development is mandatory and enforced at build time. No code without tests, no exceptions.

Rust Performance

High-performance Rust compiler with parallel processing, intelligent caching, and optimized builds.

Universal Support

Any database, storage, protocol, or hosting platform. Build once, deploy anywhere.

Strict by Default

No escape hatches from best practices. Every decision is made with quality and maintainability in mind.

Get Started in Minutes

Install Synapse Framework and start building your next application today.

1

Install the CLI

npm install -g @snps/cli
2

Create a new project

synapse init my-awesome-app
3

Start developing

cd my-awesome-app && synapse dev

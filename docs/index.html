ğŸ“Š State Management

Synapse provides a reactive state management system with immutable updates and automatic change detection.

Creating State

import { SynapseFramework } from '@snps/core'; const app = new SynapseFramework(); // Create state with initial values const state = app.createState({ user: { name: 'John', email: 'john@example.com' }, todos: [], theme: 'light' });

Updating State

// Update single property state.update('theme', 'dark'); // Update with function state.update('todos', todos => [...todos, { id: Date.now(), text: 'New todo', completed: false }]); // Update nested property state.update('user.name', 'Jane');

Subscribing to Changes

// Subscribe to specific property state.subscribe('todos', (todos) => { console.log('Todos updated:', todos); }); // Subscribe to all changes state.subscribe('*', (newState, oldState) => { console.log('State changed:', { newState, oldState }); });

State API Reference

Method Description Returns
state.get(key) Get current value of a property any
state.update(key, value) Update a property value void
state.subscribe(key, callback) Subscribe to property changes unsubscribe function
state.reset() Reset state to initial values void

ğŸ›£ï¸ Routing

Universal routing system that works on both client and server with support for dynamic routes, guards, and middleware.

Basic Routing

import { SynapseFramework } from '@snps/core'; const app = new SynapseFramework(); // Add routes app.addRoute('/', () => 'Home Page'); app.addRoute('/about', () => 'About Page'); app.addRoute('/contact', () => 'Contact Page'); // Dynamic routes app.addRoute('/users/:id', (params) => `User: ${params.id}`); app.addRoute('/posts/:year/:month', (params) => `Posts for ${params.year}/${params.month}` );

Route Guards

// Authentication guard app.addGuard('/dashboard', (user) => user?.isAuthenticated); // Role-based guard app.addGuard('/admin', (user) => user?.role === 'admin'); // Custom guard app.addGuard('/profile', (user, params) => { return user?.id === params.userId; });

Navigation

// Programmatic navigation app.navigate('/dashboard'); // Navigate with state app.navigate('/users/123', { tab: 'profile' }); // Back/forward app.goBack(); app.goForward(); // Get current route const currentRoute = app.getCurrentRoute();

ğŸ§ª Testing

TDD is mandatory in Synapse. Every piece of code must have accompanying tests with 100% coverage requirement.

Writing Tests

import { describe, it, expect } from '@snps/testing'; describe('Counter Component', () => { it('should increment count when button clicked', () => { const counter = new Counter({ initialCount: 0 }); counter.increment(); expect(counter.getCount()).toBe(1); }); it('should reset count to zero', () => { const counter = new Counter({ initialCount: 5 }); counter.reset(); expect(counter.getCount()).toBe(0); }); });

Running Tests

# Run all tests synapse test # Run tests in watch mode synapse test --watch # Run specific test file synapse test src/components/Counter.test.ts # Run with coverage synapse test --coverage

Test Coverage

Synapse enforces 100% test coverage. Build will fail if coverage is below 100%.
